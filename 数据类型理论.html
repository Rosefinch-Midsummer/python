函数参数的传递

def Swap(x,y):  
    tmp = y  
    y = x  
    x = tmp  
a = 4  
b = 5  
Swap(a,b)  
print(a,b)#4 5

a，b指向的地址对应的值没有改变。

def Swap(x,y):  
    tmp = y[0]  
    y[0] = x[0]  
    x[0] = tmp  
a = [4,5]  
b = [6,7]  
Swap(a,b)  
print(a,b)#[6, 5] [4, 7]

应用例题：校门外的树

样例输入：

500 3
150 300
100 200
470 471

样例输出：298

一共501棵树

自己实现的代码如下：

nums = input().split()  
total, n = int(nums[0]), int(nums[1])  
ls = [x for x in range(total)]  
leftover = []  
max, min = 0, 500  
for i in range(n):  
    s = input().split()  
    left, right = int(s[0]),int(s[1])  
    for m in range(left, right + 1):  
        if ls[m-1] not in leftover:  
            leftover.append(ls[m - 1])  
print(501-len(leftover))

列表的朴素排序——选择排序

def SelectionSort(a):  
    n = len(a)  
    for i in range(n-1):  
        k = i  
        for j in range(i+1,n):  
            if a[i] > a[j]:  
                a[i], a[j] = a[j], a[i]  
a = [5,4,3,2,1]  
SelectionSort(a)  
print(a) #[1, 2, 3, 4, 5]
